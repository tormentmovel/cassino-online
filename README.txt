Torment Casino Online – Enterprise Build
========================================
Este pacote contém a estrutura completa do Cassino Online Torment.
Todos os módulos estão organizados em camadas:
  • core/         – engine, pagamentos, segurança
  • games/pg/     – jogos baseados em PG‑Gaming
  • public/       – frontend (PHP + JS)
  • migrations/   – scripts SQL
  • install/      – instalador automático
  • docs/         – documentação

Para detalhes de instalação consulte docs/INSTALL_PTBR.md


Inclui pacote de assets de jogos PG‑Gaming (~120 MB) em public/assets/data/game_assets.pkg.
